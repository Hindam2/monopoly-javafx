# Monopoly JavaFX Game

A fully functional Monopoly desktop game built using Java and JavaFX.

## Features

- Turn-based game engine
- Property ownership and rent system
- Dice rolling system
- Jail and special cards logic
- Multithreading using ExecutorService
- Daemon threads for background processing
- Game state persistence using Java Serializable
- Game logging system

## Technologies

- Java
- JavaFX
- Maven
- Multithreading & Concurrency
- Object-Oriented Programming
